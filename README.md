# 使い方
## square-cloud.py
square-cloud.pyと同じ階層にXのアーカイブデータの「tweets.js」を配置。<br>
ターミナルから「python3 square-cloud.py」で実行<br>

## silhouette_cloud.py
基本の使い方はsquare-cloud.pyと同じ<br>
ただし、こちらの場合はシルエット画像が必要。<br>
背景色：白　シルエットの色：黒の画像をsilhouette_cloud.pyとtweet.jsを同じ階層に配置。<br>
「python3 silhouette_cloud.py」で実行。<br>

## 出来上がる画像
square-cloud.pyでは四角形の、<br>
silhouette_cloud.pyでは指定した画像のシルエットの形に合わせてワードが配置された画像が作られます。
